    "# Stock Data Analysis Dashboard\n",
    "\n",
    "## Project Overview\n",
    "\n",
    "This project is a **Streamlit-based interactive web application** called **Stock Data Analysis Dashboard**, developed for **ACC102 Track 4**.  \n",
    "\n",
    "It provides a simple, visual way to explore stock market data — including price trends, key statistics, and interactive charts — making it easy for business students and beginner investors to understand financial data without writing any code themselves.\n",
    "\n",
    "This project is intended for educational and descriptive analysis only. It does not provide investment advice or predict future stock prices.\n",
    "\n",
    "---\n",
    "\n",
    "## Objective\n",
    "\n",
    "The main objective of this project is to use Python and Streamlit to:\n",
    "\n",
    "- build an interactive, web-based data analysis tool\n",
    "- generate simulated stock market data for demonstration purposes\n",
    "- calculate and display key financial statistics\n",
    "- visualise stock price trends with clear, professional charts\n",
    "- offer a user-friendly interface that requires no coding knowledge\n",
    "\n",
    "---\n",
    "\n",
    "## Target Users\n",
    "\n",
    "The target users of this project are:\n",
    "\n",
    "- business and finance students (ACC102 Track 4)\n",
    "- beginner investors\n",
    "- users interested in learning how Python and Streamlit can be used for financial data analysis\n",
    "\n",
    "---\n",
    "\n",
    "## Data Source\n",
    "\n",
    "The data used in this project is **simulated sample stock data** generated programmatically.\n",
    "\n",
    "Why simulated data?\n",
    "\n",
    "- Stock price data accessed through APIs (e.g., Yahoo Finance) requires internet access and API keys\n",
    "- Simulated data ensures the app runs reliably in any environment without external dependencies\n",
    "- The simulation mimics realistic price movements using random walks with normal distribution\n",
    "\n",
    "Dataset details:\n",
    "\n",
    "- Period: Last 180 days of trading data\n",
    "- Variables included:\n",
    "  - Date\n",
    "  - Open price\n",
    "  - High price\n",
    "  - Low price\n",
    "  - Close price\n",
    "  - Adjusted Close price\n",
    "  - Trading Volume\n",
    "\n",
    "---\n",
    "\n",
    "## Key Features\n",
    "\n",
    "### 1. Interactive Ticker Input\n",
    "\n",
    "Users can enter any stock ticker symbol (e.g. AAPL, MSFT, GOOGL) in the sidebar. The dashboard will display the chart and statistics for the selected stock.\n",
    "\n",
    "### 2. Data Sample Display\n",
    "\n",
    "The app displays the most recent 10 rows of the stock data in a clean, scrollable table using Streamlit's built-in data viewer.\n",
    "\n",
    "### 3. Basic Statistics\n",
    "\n",
    "Key summary statistics are calculated and displayed, including:\n",
    "\n",
    "- **Average Closing Price** — the mean closing price over the 180-day period\n",
    "\n",
    "### 4. Price Trend Chart\n",
    "\n",
    "A professional line chart is rendered showing:\n",
    "\n",
    "- the closing price trend over the last 180 days\n",
    "- a shaded area under the curve for better visual readability\n",
    "- automatically updated labels and title based on the selected ticker\n",
    "\n",
    "### 5. Clean, Wide Layout\n",
    "\n",
    "The dashboard uses Streamlit's wide layout for a clean, full-screen presentation that is easy to read and navigate.\n",
    "\n",
